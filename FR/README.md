# 🏰 L'EMPIRE DE CODE : Pack de Réussite Génie Logiciel & Informatique de Gestion (IGL, GSI...) 2026

Bienvenue sur le dépôt officiel du projet de préparation. Cette plateforme a été conçue pour offrir aux étudiants de BTS une préparation complète, structurée et 100% pratique pour les épreuves de programmation.

## 🚀 Contenu du Dépôt

### 📅 01 - Formation Intensive (Sprint 14 Jours)
Un parcours guidé jour par jour pour maîtriser le syllabus :
- **Masterclasses** : Cours scénarisés (SQL, PHP, UML, JAVA).
- **Corrections Magistrales XL** : Résolutions intégrales des Sujets 1 et 2 (Niveau Expert).

### 🛠️ 02 - Outils Interactifs
- **Lecteur Empire** : Interface HTML pour lire les cours avec rendu automatique des diagrammes **PlantUML**.
- **Serveur Local** : Script Node.js pour diffuser le contenu.

### 📦 03 - Arsenal CRUD Ultra-Rapide
Une bibliothèque de code "Prête-à-copier" pour l'examen :
- **Niveau 1 à 4** : Du simple formulaire au système E-commerce complet.
- **Gestion par Table** : Chaque entité (Client, Produit, Inscription) possède son module CRUD dédié.
- **Sécurité** : Protection contre les doublons et injections SQL via PDO.

### 📚 04 - Fiches de Révision Langages
Fiches académiques exhaustives pour :
- **Langage C** (Programmation Structurée).
- **Langage C++** (Transition POO).
- **Java** (POO Avancée & Exceptions).
- *Inclus : Un document maître sur les concepts universels (Algorithme, Variable, Encapsulation).*

## 🛠️ Installation & Usage

1. **Cloner le projet** :
   ```bash
   git clone https://github.com/VOTRE_NOM/NOM_DU_REPO.git
   ```
2. **Importer les bases de données** :
   Chaque dossier dans `04_ARSENAL_CRUD_ULTRA_RAPIDE` contient un fichier `schema.sql` à importer dans phpMyAdmin.
3. **Lancer le lecteur** :
   Ouvrez `02_OUTILS_INTERACTIFS/LECTEUR_EMPIRE_19.5.html` dans votre navigateur ou lancez le serveur `server.js`.
