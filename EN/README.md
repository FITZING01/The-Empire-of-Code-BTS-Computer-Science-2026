# 🏰 THE CODE EMPIRE: BTS ECMN 2026 Success Pack

Welcome to the official preparation project repository. This platform was designed to provide BTS students with complete, structured, and 100% practical preparation for programming exams.

## 🚀 Repository Content

### 📅 01 - Intensive Training (14-Day Sprint)
A day-by-day guided journey to master the syllabus:
- **Masterclasses**: Scripted courses (SQL, PHP, UML, JAVA).
- **XL Master Corrections**: Full resolutions of Subject 1 and 2 (Expert Level).

### 🛠️ 02 - Interactive Tools
- **Empire Reader**: HTML interface to read courses with automatic **PlantUML** diagram rendering.
- **Local Server**: Node.js script to serve content.

### 📦 03 - Ultra-Fast CRUD Arsenal
A "Ready-to-copy" code library for the exam:
- **Level 1 to 4**: From simple forms to complete E-commerce systems.
- **Table-based Management**: Each entity (Client, Product, Registration) has its dedicated CRUD module.
- **Security**: Protection against duplicates and SQL injections via PDO.

### 📚 04 - Language Revision Sheets
Exhaustive academic sheets for:
- **C Language** (Structured Programming).
- **C++ Language** (OOP Transition).
- **Java** (Advanced OOP & Exceptions).
- *Included: A master document on universal concepts (Algorithm, Variable, Encapsulation).*

## 🛠️ Installation & Usage

1. **Clone the project**:
   ```bash
   git clone https://github.com/YOUR_NAME/REPO_NAME.git
   ```
2. **Import databases**:
   Each folder in `03_ARSENAL_CRUD_ULTRA_RAPIDE` (renamed from 04 in some contexts) contains a `schema.sql` file to import into phpMyAdmin.
3. **Launch the reader**:
   Open `02_OUTILS_INTERACTIFS/LECTEUR_EMPIRE_19.5.html` in your browser or run the `server.js` server.
