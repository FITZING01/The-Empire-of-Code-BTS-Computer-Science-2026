# WEB JS MASTERCLASS FROM ZERO TO HERO: Interactivity and Client Logic

## Scenario: The Magic of the Persistent Cart

In our last lesson, we generated the shop with PHP. But what happens if the user clicks "Add to Cart"? If we reload the page on every click via PHP, the user experience will be slow and frustrating. We want a modern, responsive interface where the cart updates instantly, and most importantly, stays saved if the customer closes their browser.
This is JavaScript's mission.

### 1. LocalStorage: The Browser's Memory

The web is "stateless." A web page remembers nothing between two reloads. To overcome this, we use the Web Storage API, specifically `localStorage`.
- **Advantage**: Allows storing up to 5 MB of data per domain, directly in the client's browser.
- **Persistence**: Data survives browser closure (unlike `sessionStorage`).
- **Limitation**: LocalStorage can store ONLY character strings.

### 2. JSON Format: The Esperanto of the Web

Since localStorage only stores text, how do we save an array of objects (our cart)? With JSON (JavaScript Object Notation).
- `JSON.stringify(object)`: Transforms a JS object/array into a character string (for writing).
- `JSON.parse(string)`: Transforms a character string into a JS object/array (for reading).

### 3. The Power of Array Iterators: find and reduce

Junior developers write endless `for` loops to iterate through arrays. Seniors use high-level methods that are expressive and immutable.
- `Array.find()`: Iterates through the array and returns the FIRST element that satisfies a condition. Perfect for checking if a jersey is already in the cart.
- `Array.reduce()`: Iterates through the array and "reduces" all values into a single one (e.g., summing prices or getting the total item count).

### 4. The Masterpiece Code: The Cart Engine (cart.js)

Here is the complete implementation of the cart system in pure JavaScript (Vanilla JS). No need for jQuery. This code is modular, robust, and professionally commented.

```javascript
/**
 * @file cart.js
 * @description Complete management of the shopping cart via LocalStorage for JerseyPro.
 * This script attaches to buttons generated by PHP and manages the state on the client side.
 */

// Module pattern to encapsulate cart logic and avoid polluting the global space
const CartManager = (function() {
    
    // Key used in LocalStorage
    const STORAGE_KEY = 'jerseypro_cart';

    /**
     * @function readCart
     * @returns {Array} The array of products in the cart
     */
    const readCart = () => {
        const cartJSON = localStorage.getItem(STORAGE_KEY);
        // If the cart doesn't exist, we return an empty array
        if (!cartJSON) return [];
        
        try {
            return JSON.parse(cartJSON);
        } catch (error) {
            console.error("LocalStorage corruption error", error);
            return []; // Security fallback
        }
    };

    /**
     * @function saveCart
     * @param {Array} cart 
     */
    const saveCart = (cart) => {
        localStorage.setItem(STORAGE_KEY, JSON.stringify(cart));
        updateUI(); // We update the interface on every save
    };

    /**
     * @function addProduct
     * @param {Object} product - {id, name, price}
     */
    const addProduct = (product) => {
        const cart = readCart();
        
        // Using .find(): Does the product already exist?
        const existingProduct = cart.find(item => item.id === product.id);

        if (existingProduct) {
            // If it exists, we just increment the quantity
            existingProduct.quantity += 1;
        } else {
            // Otherwise, we add it with an initial quantity of 1
            cart.push({
                ...product,
                quantity: 1
            });
        }

        saveCart(cart);
        
        // UX Feedback (Micro-interaction)
        showToast(`"${product.name}" has been added to the cart!`);
    };

    /**
     * @function updateUI
     * @description Updates the counter in the navigation bar
     */
    const updateUI = () => {
        const cart = readCart();
        const badge = document.getElementById('cart-counter');
        
        if (badge) {
            // Using .reduce(): Sum of all quantities
            // reduce((accumulator, currentValue) => ..., initialValue)
            const totalItems = cart.reduce((total, item) => total + item.quantity, 0);
            
            // Subtle animation to catch the eye
            badge.style.transform = 'scale(1.2)';
            badge.textContent = totalItems;
            setTimeout(() => { badge.style.transform = 'scale(1)'; }, 200);
        }
    };

    /**
     * @function showToast
     * @description Dynamic creation of a notification (essential UX feedback)
     */
    const showToast = (message) => {
        // In production, we would use a Bootstrap Toast component
        // Here, custom light implementation
        const toast = document.createElement('div');
        toast.textContent = message;
        toast.style.cssText = `
            position: fixed;
            bottom: 20px;
            right: 20px;
            background-color: #198754;
            color: white;
            padding: 1rem;
            border-radius: 8px;
            box-shadow: 0 4px 6px rgba(0,0,0,0.1);
            z-index: 1050;
            transition: opacity 0.3s;
        `;
        document.body.appendChild(toast);
        
        setTimeout(() => {
            toast.style.opacity = '0';
            setTimeout(() => toast.remove(), 300);
        }, 3000);
    };

    // Public API exposed by the module
    return {
        init: () => {
            // UI initialization on load
            updateUI();
            
            // Event listeners on all "Add to cart" buttons
            const buttons = document.querySelectorAll('.btn-add-cart');
            buttons.forEach(btn => {
                btn.addEventListener('click', (e) => {
                    const id = parseInt(e.target.dataset.id);
                    const name = e.target.dataset.name;
                    const price = parseFloat(e.target.dataset.price);
                    
                    addProduct({ id, name, price });
                });
            });
        }
    };
})();

// Launch the mechanics once the DOM is fully loaded
document.addEventListener('DOMContentLoaded', CartManager.init);
```

### Senior's Lesson
Using an IIFE (Immediately Invoked Function Expression) to create a `CartManager` "Module" is a key architectural practice. It prevents your `readCart` or `STORAGE_KEY` variables from interfering with other scripts on the page. Also note the importance of UX Feedback (the `showToast` function and the badge animation). An action without feedback is an action perceived as failed by the user.
